### Устанока GIT в Linux
---

<br>

Проверяем наличие установленного Git, выполняем в терминале:

`$ git --version`

![lin_install](./assets/lin_install.png)


Если Git не установлен, выполняем в терминале команду:

`$ sudo apt install git`
